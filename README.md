# GNSS-Racing-Telemetry

How to run

1. Create virtual envioroment:
  (Windows)
  python -m venv .venv

2. Activate it:
  (Windows) .venv\Scripts\activate
  (Mac or Linux) source .venv/bin/activate

3. (Mac or Linux)
  source .venv\bin\activate

4. Install libraries:
  pip install  -r requirements.txt

5. Run:
  python -m src.main
